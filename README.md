# Halloween Workshop 2021
This repository contains code and other assets used in the 2021 WiFi Nugget Halloween workshop.

## Code

**Binary Files**   

The following binary files can be flashed and immediately run on the WiFi Nugget, which are modified spooky version of the code available [here](https://github.com/HakCat-Tech/WiFiNugget).
- **Halloween-HaxxDetector.bin** - program to detect and visualize WiFi attacks
- **Halloween-Deauther.bin** - WiFi hacking & monitoring multitool by Spacehuhn
